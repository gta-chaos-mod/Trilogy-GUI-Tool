GTA Trilogy Chaos Mod Token Generator

To get started, head to the Twitch Developer Dashboard and create a new Application with the category set to "Website Integration".

Name it whatever you want and set the OAuth Redirect URL to https://chaos.lord.moe/

To revoke access, disconnect the app you created from your Twitch settings.

(Technical: This application uses the implicit grant flow for the Twitch API to retrieve your token. This means that your token is only ever visible to your browser and not our server)
